\# OCR Document Reconstructor



Converts scanned/photographed document images into structured `.docx` files using PaddleOCR + PP-StructureV3, preserving headings, paragraphs, and tables instead of producing a flat text dump.



\## Status: Work in progress (Day 3 of build)



\## Stack

\- PaddleOCR / PP-StructureV3

\- python-docx (reconstruction)

\- OpenCV (preprocessing)
